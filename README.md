# FarmVille

### Açıklama
.Net 6.0 ile hazırlanmış MongoDb veritabanı kullanılan dinamik bir mini web sitesi projesi.

### Kullanılan Teknolojiler
- .Net 6.0
- MVC
- MongoDB
- Html Css Bootstrap
